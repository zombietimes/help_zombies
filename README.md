## Usage
### Set up
```sh
# Ubuntu commands.
git clone https://github.com/zombietimes/help_zombies.git
cd help_zombies
sh ./ubuntuCmd_setupHelp_zombies.sh
```
![help_zombies_0000.png]()  

### Run
```sh
# Ubuntu commands.
sh ~/dapps/tools/help_root.sh
```
